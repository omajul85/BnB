###Instructions
To deploy local version
```
$ git clone https://github.com/omajul85/BnByte4.git
$ cd BnBytes
$ bundle # Installs all dependencies
$ rackup # Runs the application locally
```

To access live site: https://bnbyte4.herokuapp.com
